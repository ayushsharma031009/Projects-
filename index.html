Welcome to MelodiFy

Experience the joy of seamless music streaming with MelodiFy! Our platform allows you to explore, play, and enjoy your favorite tracks anytime, anywhere. Whether you're in the mood for some soothing melodies or an upbeat jam session, MelodiFy has got you covered.

With a user-friendly interface and a vast collection of songs, MelodiFy is designed to enhance your music experience. Dive in and let the rhythm take over!

How I Built This Website

This project was built using Tailwind CSS for styling and the Spotify API via RapidAPI for fetching music data. The website allows users to search for songs or retrieve the top 10 songs of an artist.

Through this project, I demonstrated my ability to work with APIs, handle data efficiently, and implement JavaScript functionalities.
